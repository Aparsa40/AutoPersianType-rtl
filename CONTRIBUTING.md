# Contributing

خوش‌آمدید! از علاقه‌مندی شما برای کمک به AutoPersianType متشکریم.

- لطفاً قبل از ارسال Pull Request، یک Issue باز کنید تا بحث انجام شود.
- فایل‌های هسته‌ای (`server/`, `package.json`, `drizzle.config.ts`, `vite.config.ts`) تحت حفاظت هستند و تغییر آن‌ها نیازمند تایید نگهدارندگان (دیدن `.github/CODEOWNERS`) است.
- برای مسائل کوچک، مستقیماً Fork کنید، تغییر دهید و Pull Request ارسال کنید.

تشکر! ✨
