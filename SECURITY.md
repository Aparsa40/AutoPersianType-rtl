# Security Policy / سیاست امنیتی

AutoPersianType takes security seriously. If you discover a vulnerability, do NOT create a public issue.
AutoPersianType امنیت را جدی می‌گیرد. اگر آسیب‌پذیری پیدا کردید، **لطفاً Issue عمومی باز نکنید**.

---

## Reporting / گزارش

Contact the maintainers via email: [parssense84@gmail.com](mailto:parssense84@gmail.com)
با نگهدارندگان از طریق ایمیل تماس بگیرید: [parssense84@gmail.com](mailto:parssense84@gmail.com)

Include:

- Steps to reproduce / مراحل بازتولید
- Affected versions / نسخه‌های آسیب‌پذیر
- Suggested mitigations / راه‌حل‌های پیشنهادی

---

## Disclosure / افشا

Once the issue is fixed, it will be disclosed in coordination with the reporter.
پس از رفع مشکل، افشا با هماهنگی گزارش‌دهنده انجام خواهد شد.
