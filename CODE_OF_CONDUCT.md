# Code of Conduct / قواعد رفتاری

We expect all contributors to follow basic respectful behavior. Keep discussions polite and constructive. Harassment, abuse, or hateful content will not be tolerated.
ما انتظار داریم همه مشارکت‌کنندگان رفتار محترمانه داشته باشند. بحث‌ها باید مودبانه و سازنده باشد. هرگونه آزار، سوءرفتار یا محتوای نفرت‌انگیز پذیرفته نیست.

If you experience or witness unacceptable behavior, please open an issue addressed to the maintainers or contact the project admins.
اگر شما تجربه یا مشاهده رفتار نامناسب داشتید، لطفاً یک Issue به نگهدارندگان پروژه ارسال کنید یا با مدیران پروژه تماس بگیرید.

---

## Summary / خلاصه

- Respect others / احترام به دیگران
- Constructive discussions / بحث‌های سازنده
- Report issues / گزارش رفتار نامناسب
